    A base de dados · Wagner

    Tudo a partir de dados públicos

    A premissa do ALIME é usar só fontes oficiais e abertas — por isso a mesma análise se repete em qualquer município de pequeno porte, sem pesquisa de campo cara.

    • INEP matrículas escolares → polos de atração

    A demanda · Wagner

    Dos dados à matriz origem-destino

    02Vetores de atração

    03Modelo gravitacional

    distribui as viagens · atrito espacial β = 1,50

    A ferramenta · Luiz

    O ALIME por trás dos números

    Um simulador web de apoio à decisão preliminar para municípios de pequeno porte (até ~20 mil hab) — onde falta dado e verba para um estudo completo.

    • Genérico: serve a qualquer cidade — Matias Barbosa é só o caso demonstrativo.
    • Aberto: Python · Streamlit · OSMnx · NetworkX, rodando no navegador.
    • Exploratório: não substitui EVTEA, contagem, O-D domiciliar nem microssimulação.
